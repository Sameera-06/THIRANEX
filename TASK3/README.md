# Exploratory Data Analysis (EDA) Project

## Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on a sales dataset to uncover hidden patterns, trends, and relationships. The analysis was conducted using Python and various data visualization techniques to gain meaningful business insights.

---

## Objectives

* Understand the structure of the dataset
* Generate statistical summaries
* Explore relationships between variables
* Identify trends and patterns
* Analyze correlations among features
* Present findings through visualizations

---

## Dataset Description

The dataset contains sales-related information including:

* Order Details
* Product Categories
* Sales
* Profit
* Customer Segments
* States and Regions

The dataset was analyzed to understand factors affecting sales and profitability.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## Exploratory Data Analysis Steps

### 1. Data Inspection

* Loaded the dataset
* Examined column names and data types
* Checked dataset dimensions

### 2. Data Cleaning

* Identified missing values
* Verified data consistency
* Prepared data for analysis

### 3. Statistical Analysis

* Generated descriptive statistics
* Examined distributions of numerical variables
* Analyzed central tendencies and variability

### 4. Data Visualization

Created multiple visualizations including:

* Correlation Heatmap
* Sales by Category
* Profit by Category
* Sales Distribution Histogram
* Top States by Sales
* Customer Segment Analysis

### 5. Correlation Analysis

* Evaluated relationships between numerical variables
* Identified factors influencing sales and profit

---

## Key Findings

* Technology products generated the highest sales.
* Sales and profit showed a positive correlation.
* A small number of states contributed significantly to overall revenue.
* Customer segments displayed different purchasing patterns.
* Most sales transactions were concentrated within a moderate sales range.
